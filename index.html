▷ 材料

  • かぶの葉適量

▷ 手順

▷ メモ

魚

ぶり大根家族推し

▷ 材料（抜粋）

  • ぶり適量
  • 大根1cm輪切り
  • みりん : 酒 : しょうゆ5 : 3 : 1
  • しょうが適量

▷ 手順

  1. ぶりに塩→ラップして冷蔵庫20〜30分
  2. 大根1cm輪切り→下茹で→ザルにあげる
  3. ぶりを半分に切り湯通し→すぐ冷水へ
  4. 煮汁（みりん150:酒90:しょうゆ30）を中火で煮立て材料を入れる
  5. 沸いたら1回ひっくり返し一度取り出す→大根だけ煮含める
  6. 魚を戻してしょうがも入れ、とろみがついたら煮汁をからめて完成

▷ メモ

煮汁比率 みりん:酒:しょうゆ = 5:3:1。しらがねぎを添えると◎

煮込み・カレー兼用 肉

鶏ももの煮込み＆カレー

▷ 材料

  • 鶏もも肉2枚
  • 玉ねぎ2〜3個
  • 塩小さじ1/2
  • 水250cc
  • カレー粉（カレーにする場合）小さじ2〜3

▷ 手順（概要）

  1. 玉ねぎを横半分に切る
  2. 油なしで鶏もも肉を焼き、両面こんがり焼き目をつけて鍋へ
  3. 出た脂で玉ねぎも両面焼き、鶏肉の上に敷き詰める
  4. フライパンの旨みを溶かした水250ccと塩を加え蓋をして中火45分煮込む
  5. そのまま器に盛り、肉をほぐしながら食べる

▷ メモ

焦げ目しっかりがポイント。カレーにする場合はほぐした後にカレー粉を加えてひと煮立ち。
煮込み・カレー兼用 肉

鶏ももの煮込み＆カレー

▷ 材料

  • 鶏もも肉2枚
  • 玉ねぎ2〜3個
  • 塩小さじ1/2
  • 水250cc
  • カレー粉（カレーにする場合）小さじ2〜3

▷ 手順

  1. 玉ねぎを横半分に切る
  2. フライパンに油を敷かず鶏もも肉を焼く。両面こんがり焼き目がついたら大きめの鍋に移す
  3. 出た脂で玉ねぎの両面に焼き目をつけ、鶏肉の上に敷き詰める
  4. フライパンの旨みを溶かした水250ccを鍋に加え、塩小さじ1/2を入れる
  5. 蓋をして中火で45分煮込む
  6. そのまま器に盛り、鶏肉をほぐしながら食べる（煮込み完成）
  7. カレーにする場合：ヘラでほぐしカレー粉 小さじ2〜3を加えよく混ぜ、少し煮込んで完成

▷ メモ

焦げ目しっかりがポイント。油なしで焼くことで余分な脂が出て旨みも凝縮。カレー粉の量はお好みで調整を。

キャロットケーキお気に入り

▷ 材料（全量）

  • 人参（中）蒸後 約150g1本（160g〜）
  • 白みそ20g
  • みりんシロップ10g
  • 塩ひとつまみ（0.3g）
  • 米粉→ 150gに増量も可120g
  • ベーキングパウダー→ 5gに増量も可4g

  • シナモンなど適量

  4. 全卵を入れて3回目の攪拌、全体をよく混ぜる
  5. 型に入れて空気を抜き180℃ · 40分で焼く

米粉120→150g、BP 4→5gで調整可。シナモンなどスパイスはお好みで。
